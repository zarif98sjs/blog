Grammar in C (a programming use case)

Regular Grammar

Limit production rules to have exactly one nonterminal on LHS and at most one nonterminal and terminal on RHS

Context Free Grammar

Limit production rules to have exactly one nonterminal on LHS, but anything on RHS

Context Sensitive Grammar

Ambiguous String

  • has 2 or more possible parse tree

Ambiguous Grammar

  • generates ambiguous strings
  • no algorithm to identify / remove ambiguity
  • source of ambiguity
    • no way to enforce operator precedence
    • no way to enforce operation from a particular side (left or right)
  • removing ambiguity

Inherently ambiguous language

  • all grammars are ambiguous

Unambiguous language

  • if at least one grammar is unambiguous
